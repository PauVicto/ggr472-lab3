<details>
<summary>
Summary of GGR472 Lab 3 MAPBOX GLJS + Legend
</summary>
This lab is an interactive web map created using Mapbox GLJS that showcases Vic's favoriate cafes, restaurants, and study spots in downtown Toronto. With the utilization GeoJSOn data integraded into this interactive map and styled utilizing Mapbox expressions and CSS. 

This map includes interactive features such as map popups, cursor changes, as well as a legend. Additionally, the "match, ['get', 'category']" expression is used to check a GeoJSON feature's property (category) and returns a a result such as an specific icon, allowing the map to apply different styles and icons based on a value from JSON. 

This map allows users to explore these locations through an interactive experience and allows users to explore locations using a geocoder search bar and location popups.
